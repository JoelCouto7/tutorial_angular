Tutorial Angular

¿Qué es AngularJS?

AngularJS es un framework de JavaScript de código abierto, mantenido por Google.

Esencialmente hace que el desarrollo y las pruebas de aplicaciones web sean más fáciles.

Para integrarlo en tu web sólo necesitas conocer su sintaxis y enlazarlo al código HTML de la siguiente forma.

<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.9/angular.min.js"></script>

AngularJS no requiere ningún tipo de instalción, debido a que se enlaza a la web mediante el código anterior.

Sobre éste tutorial

En éste tutorial de AngularJS se van a explicar diferentes funcionalidades básicas del lenguaje y cómo expande las funcionalidades de HTML, al final de cada explicación hay un breve código de ejemplo.

Cada sección de éste tutorial de AngularJS es básicamente un extracto del tutorial hecho por W3Schools.

Al final del tutorial hay un ejemplo de aplicación completa hecha con AngularJS que puede ser probada.

Primeros pasos

AngularJS expande las funcionalidades de HTML con las directivas ng:

La directiva ng-app define una aplicación AngularJS.

La directiva ng-model vincula el valor de los controles HTML (input, select, textarea) a los datos de la aplicación.

La directiva ng-bind vincula los datos de la aplicación a la vista HTML.

AngularJS Ejemplo

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.9/angular.min.js"></script>
<body>

<div ng-app="">
  <p>Name: <input type="text" ng-model="name"></p>
  <p ng-bind="name"></p>
</div>

</body>
</html>

El código anterior genera el siguiente resultado:

Name:

Explicación:

AngularJS se inicia automáticamente cuando se carga la página web.

La directiva ng-app le dice a AngularJS que el elemento "div" es el "propietario" de una aplicación AngularJS.

La directiva ng-model vincula el valor del campo de entrada al nombre de la variable de la aplicación.

La directiva ng-bind vincula el contenido del elemento "p" al nombre de la variable de la aplicación.

Expressions

AngularJS vincula datos a HTML usando Expressions.

Las expresiones de AngularJS se pueden escribir entre llaves dobles: {{ expression }}.

Las expresiones de AngularJS también se pueden escribir dentro de una directiva: ng-bind="expression".

AngularJS resolverá la expresión y devolverá el resultado exactamente donde está escrita la expresión.

Las expresiones de AngularJS son muy similares a las expresiones de JavaScript: pueden contener literales, operadores y variables.

Ejemplo

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.9/angular.min.js"></script>
<body>

<div ng-app="">
  <p>My first expression: {'{ 5 + 5 }'}</p>
</div>

</body>
</html>

Modules

Un módulo AngularJS define una aplicación.

El módulo es un contenedor para las diferentes partes de una aplicación.

El módulo es un contenedor para los controladores de la aplicación.

Los controladores siempre pertenecen a un módulo.

Creando un Módulo

Se crea un módulo utilizando la función angular.module de AngularJS

<div ng-app="myApp">...</div>

<script>

var app = angular.module("myApp", []);

</script>

Directives

AngularJS te permite ampliar HTML con nuevos atributos llamados Directives.

AngularJS tiene un conjunto de directivas integradas que ofrecen funcionalidad a sus aplicaciones.

AngularJS también te permite definir tus propias directivas.

AngularJS Directives

Las directivas AngularJS son atributos HTML extendidos con el prefijo ng-.

La directiva ng-app inicializa una aplicación AngularJS.

La directiva ng-init inicializa los datos de la aplicación.

La directiva ng-model vincula el valor de los controles HTML (input, select, textarea) a los datos de la aplicación.

Ejemplo

<div ng-app="" ng-init="firstName='John'">

<p>Name: <input type="text" ng-model="firstName"></p>
<p>You wrote: {{ firstName }}</p>

</div>

Model

La directiva ng-model vincula el valor de los controles HTML (input, select, textarea) a los datos de la aplicación.

La Directiva ng-model

Con la directiva ng-model puedes vincular el valor de un campo de entrada a una variable creada en AngularJS.

Ejemplo

<div ng-app="myApp" ng-controller="myCtrl">
  Name: <input ng-model="name">
</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
  $scope.name = "John Doe";
});
</script>

Controllers

Los controladores de AngularJS controlan los datos de las aplicaciones en AngularJS.

Los controladores de AngularJS son Objetos JavaScript normales.

AngularJS Controllers

Las aplicaciones AngularJS están controladas por controladores.

La directiva ng-controller define el controlador de la aplicación.

Un controlador es un Objeto de JavaScript, creado por un constructor de objetos en JavaScript .

AngularJS Ejemplo

<div ng-app="myApp" ng-controller="myCtrl">

First Name: <input type="text" ng-model="firstName"><br>
Last Name: <input type="text" ng-model="lastName"><br>
<br>
Full Name: {{firstName + " " + lastName}}

</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
  $scope.firstName = "John";
  $scope.lastName = "Doe";
});
</script>

Scopes

El scope es la parte de enlace entre el HTML (vista) y el JavaScript (controlador).

El scope es un objeto con las propiedades y métodos disponibles.

El scope está disponible tanto para la vista como para el controlador.

¿Cómo usar el Scope?

Cuando creas un controlador en AngularJS, la pasas el objeto $scope como un argumento:

Ejemplo

Las propiedades creadas en el controlador se pueden consultar en la vista:

<div ng-app="myApp" ng-controller="myCtrl">

<h1>{{carname}}</h1>

</div>

<script>
var app = angular.module('myApp', []);

app.controller('myCtrl', function($scope) {
  $scope.carname = "Volvo";
});
</script>

Filters

Se pueden agregar filtros en AngularJS para formatear datos.

AngularJS Filters

AngularJS proporciona filtros para transformar datos:

  • currency Formatea un número a un formato de moneda.
  • date Formatea una fecha a un formato específico.
  • filter Selecciona un subconjunto de elementos de una matriz.
  • json Formatea un objeto a una cadena JSON.
  • limitTo Limita una matriz/cadena a un número específico de elementos/caracteres.
  • lowercase Formatea una cadena en minúsculas.
  • number Formatea un número en una cadena.
  • orderBy Ordena una matriz mediante una expresión.
  • uppercase Formatea una cadena en mayúsculas.

Agregar filtros a las expresiones

Se pueden agregar filtros a las expresiones utilizando el carácter de barra vertical |, seguido de un filtro.

El filtro uppercase formatea cadenas en mayúsculas:

Ejemplo

<div ng-app="myApp" ng-controller="personCtrl">

<p>The name is {'{ lastName | uppercase }'}</p>

</div>

Services

En AngularJS puedes crear tu propio servicio o utilizar uno de los muchos servicios integrados.

¿Qué es un Servicio?

En AngularJS, un servicio es una función u objeto que está disponible y limitado a su aplicación AngularJS.

AngularJS tiene alrededor de 30 servicios integrados. Uno de ellos es el servicio $location.

El servicio $location tiene métodos que devuelven información sobre la ubicación de la página web actual.:

Ejemplo

Use the $location service in a controller:

var app = angular.module('myApp', []);
app.controller('customersCtrl', function($scope, $location) {
    $scope.myUrl = $location.absUrl();
});

Routing

El módulo ngRoute ayuda a que su aplicación se convierta en una aplicación de página única.

¿Qué es el enrutamiento en AngularJS?

Si desea navegar a diferentes páginas en su aplicación, pero también desea que la aplicación sea una SPA (Aplicación de página única), sin recarga de página, puede usar el módulo ngRoute.

El módulo ngRoute enruta su aplicación a diferentes páginas sin tener que recargar toda la aplicación.

Ejemplo:

Navigate to "red.htm", "green.htm", and "blue.htm":

<body ng-app="myApp">

<p><a href="#/!">Main</a></p>

<a href="#!red">Red</a>
<a href="#!green">Green</a>
<a href="#!blue">Blue</a>

<div ng-view></div>

<script>
var app = angular.module("myApp", ["ngRoute"]);
app.config(function($routeProvider) {
  $routeProvider
  .when("/", {
    templateUrl : "main.htm"
  })
  .when("/red", {
    templateUrl : "red.htm"
  })
  .when("/green", {
    templateUrl : "green.htm"
  })
  .when("/blue", {
    templateUrl : "blue.htm"
  });
});
</script>
</body>

Application

Es hora de crear una verdadera aplicación AngularJS.

Crear una lista de compras

Utilicemos algunas de las funciones de AngularJS para crear una lista de compras, en la que se pueden agregar o quitar artículos:

My Shopping List

  • {{x}}×


Aplicación Explicada

Paso 1. Comenzando:

Comience por crear una aplicación llamada myShoppingList y agréguele un controlador llamado myCtrl.

El controlador agrega una matriz llamada products al $scope actual.

En HTML, usamos la directiva ng-repeat para mostrar una lista usando los elementos de la matriz.

Ejemplo

Hasta ahora hemos creado una lista HTML basada en los elementos de una matriz:

<script>
var app = angular.module("myShoppingList", []);
app.controller("myCtrl", function($scope) {
  $scope.products = ["Milk", "Bread", "Cheese"];
});
</script>

<div ng-app="myShoppingList" ng-controller="myCtrl">
  <ul>
    <li ng-repeat="x in products">{{x}}</li>
  </ul>
</div>

Paso 2. Agregar elementos:

En el HTML, agregue un campo de texto y vincúlelo a la aplicación con la directiva ng-model.

En el controlador, cree una función llamada addItem y use el valor del campo de entrada addMe para agregar un elemento a la matriz products.

Agregue un botón y asígnele una directiva ng-click que ejecutará la función addItem cuando se haga clic en el botón.

Ejemplo

Ahora podemos agregar elementos a nuestro carrito de compras lista:

<script>
var app = angular.module("myShoppingList", []);
app.controller("myCtrl", function($scope) {
  $scope.products = ["Milk", "Bread", "Cheese"];
  $scope.addItem = function () {
    $scope.products.push($scope.addMe);
  }
});
</script>

<div ng-app="myShoppingList" ng-controller="myCtrl">
  <ul>
    <li ng-repeat="x in products">{{x}}</li>
  </ul>
  <input ng-model="addMe">
  <button ng-click="addItem()">Add</button>
</div>

Paso 3. Eliminar elementos:

También queremos poder eliminar elementos de la lista de compras.

En el controlador, crea una función llamada removeItem, que toma como parámetro el índice del elemento que deseas eliminar.

En el HTML, crea un elemento <span> para cada elemento y dales una directiva ng-click que llame a la función removeItem con el $index actual.

Ejemplo

Ahora podemos eliminar elementos de nuestra lista de compras:

<script>
var app = angular.module("myShoppingList", []);
app.controller("myCtrl", function($scope) {
  $scope.products = ["Milk", "Bread", "Cheese"];
  $scope.addItem = function () {
    $scope.products.push($scope.addMe);
  }
  $scope.removeItem = function (x) {
    $scope.products.splice(x, 1);
  }
});
</script>

<div ng-app="myShoppingList" ng-controller="myCtrl">
  <ul>
    <li ng-repeat="x in products">
      {{x}}<span ng-click="removeItem($index)">&times;</span>
    </li>
  </ul>
  <input ng-model="addMe">
  <button ng-click="addItem()">Add</button>
</div>

Paso 4. Control de errores:

La aplicación tiene algunos errores, como por ejemplo, si intentas agregar el mismo artículo dos veces, la aplicación se bloquea. Además, no debería permitirse agregar artículos vacíos.

Lo solucionaremos comprobando el valor antes de agregar nuevos artículos.

En el HTML, agregaremos un contenedor para mensajes de error y escribiremos un mensaje de error cuando alguien intente agregar un artículo existente.

Ejemplo

Una lista de compras, con la posibilidad de escribir mensajes de error:

<script>
var app = angular.module("myShoppingList", []);
app.controller("myCtrl", function($scope) {
  $scope.products = ["Milk", "Bread", "Cheese"];
  $scope.addItem = function () {
    $scope.errortext = "";
    if (!$scope.addMe) {return;}
    if ($scope.products.indexOf($scope.addMe) == -1) {
      $scope.products.push($scope.addMe);
    } else {
      $scope.errortext = "The item is already in your shopping list.";
    }
  }
  $scope.removeItem = function (x) {
    $scope.errortext = "";
    $scope.products.splice(x, 1);
  }
});
</script>

<div ng-app="myShoppingList" ng-controller="myCtrl">
  <ul>
    <li ng-repeat="x in products">
      {{x}}<span ng-click="removeItem($index)">&times;</span>
    </li>
  </ul>
  <input ng-model="addMe">
  <button ng-click="addItem()">Add</button>
  <p>{{errortext}}</p>
</div>

Agregue la hoja de estilo W3.CSS e incluya las clases adecuadas en toda la aplicación; el resultado será el mismo que la lista de compras en la parte superior de esta página.
